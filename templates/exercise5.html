Overview

  • So apparently, Waldo's gone missing again!
  • Witnesses say they saw him hiding inside of Python arrays!
  • In this exercise, you will implement a program that looks at lists and dictionaries, and ultimately tell the user: "Where's Waldo?"

Instructions

  • In Python, create a program that searches for Waldo.
  • The program will take one input, people, which is a list of dictionaries.
  • Inside each dictionary is a name, a city, and a disguise.
  • You must print find Waldo's information and print it out as a list!

Test Cases

  • You must print Waldo's name, city, and disguise on the terminal line as a list.
  • The outputs are case-sensitive.
  • Your may assume that there will be no duplicate names, and therefore, only one Waldo.
