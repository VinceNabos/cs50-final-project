Lesson 1: Variables, Data Types, Operators

Getting Started with Python

  • At Syntax, exercises can be done without installing anything.
  • However, it may be in your best interest to try out code as we go through different concepts!
  • To run Python, you first have to check if Python is installed. Type the following commands to check for Python: 
                    
    (For Windows, use command prompt)
    C:\Users\[Your Name]>python --version

    (For Linux or Mac, use the command line or terminal respectively)
    python --version
  • If python is not installed, you can download it from: https://www.python.org/
  • In any text editor (eg. Notepad), you can start writing your own Python code! Let us write our first Python file, 'hello.py': 
                    
    print('hello, world')
  • Save your file, go back to the command line, navigate to the location of your file, and run the following: 
                    
    python hello.py
  • The output should be: 
                    
    hello, world
  • Congrats! You have written your first Python program!
  • You can also use Visual Studio Code, a very popular IDE, to create Python programs.
  • Check out their Python tutorial here!

Functions (Overview)

  • What you just wrote was a print program. print() is a function that simply displays output to the screen.
  • Python comes built-in with these functions; tools to perform certain tasks that we can call whenever we need them.
  • To call a function, simply type its name and the inputs inside parentheses! Recall that our print function took 'hello world' as an input and "printed" it to the screen!
  • We will encounter different types of functions later on (and hey, you'll even create your own!) once we learn the basic fundamentals of Python.

Variables

  • Variables are simply containers for storing data values. Say you want to represent the number five (5) in a variable: 
                    
    # Tip: You can write comments in your code using hashes!
    x = 5
    print(x)
  • Your first instinct might be that the print() function prints out 'x'. However, in your code, x is now a variable containing the number five (5)! Thus, the output of your program is: 
                    
    # Output
    5
  • Variables are case sensitive. In the ccode below, the variables do not have the same values even though they are essentially the same letter. 
                    
    x = 5
    X = 404

    print(x)
    print(X)

    # Output
    5
    404
  • A variable can be changed in the middle of your program. Python reads code from top to bottom, so be careful when changing the values of your variables! 
                    
    x = 5

    # Random code goes here

    x = 'John'
    print(x)

    # Output
    John
  • Wait, notice that? The variable x was assigned a text value instead of a number! Variables can be assigned different data types. But what exactly are those?

Data Types

  • Variables can store different data types, and each type can do different things. The following are some of Python's built-in data types that you will see in this course: 
                    
    # Texts         # Numerics          # Boolean
    str             int, float          bool

    # Sequences     # Maps              # None 
    list, range     dict                None
  • You can get the data type of a variable by using the type() function. 
                    
    x = 0
    print(type(x))

    # Output
    <class 'int'>
  • The most fundamental of these data types are strings, ints, floats, booleans, and the 'None' value. All programming languages have these data types by default, though some may have some weird implementations.
  • We can do all sorts of stuff with these data types! But first, let us understand what each one stores.
  • Strings (str) are esentially a sequence of characters used to represent text. Think of any word. When that word is assigned to a variable, the variable is now a string. 
                    
    # Use single or double quotes to denote a string! Either work, just be consistent!
    # This is important as Python, like most languages, recognize strings by the quotes.
    x = 'David'
    y = "cat"

    print(type(x))
    print(type(y))

    # Output
    <class 'str'>
    <class 'str'>
  • Numbers can be represented by integers (int) or floats (float). Integers are the whole numbers you know and love while floats have decimal places, even if they are trailing zeros. 
                    
    x = 1
    y = 1.0

    print(type(x))
    print(type(y))

    # Output
    <class 'int'>
    <class 'float'>
  • The None data type represents, well, nothing! To be clear, it represents the abscence of a value or a null state. It could also represent the emptiness of a list (more on that soon!). 
                    
    x = None
    print(type(x))

    # Output
    <class 'NoneType'>
  • A boolean can hold only one of two possible values: True or False. It's as simple as it sounds! When something is true, a boolean says it's true. If it isn't, then a boolean says it's false. 
                    
    # In Python, booleans are capitalized!
    x = True

    print(type(x))
    print(5 < 3) # More on operators soon!

    # Output
    <class 'bool'>
    False
  • If you've looked at your sidebar, you'll see lists and dictionaries, two more Python data types. We'll get to those soon! Python has a bunch more, but it'll be up to you to explore those!

Operators

  • Operators are used to perform operations on variables and values.
  • Arithmetic operators are used to perform common mathematical operations: 
                    
    x = 15
    y = 5

    print(x + y)    # +, addition
    print(x - y)    # -, subtraction
    print(x * y)    # *, multiplication
    print(x / y)    # /, division (returns a float)
    print(x // y)   # //, floor division (returns an integer, rounded down)
    print(x % y)    # %, module (returns a remainder)

    # Output
    20
    10
    75
    3.0
    3
    0
  • The equal sign is Python's assignment operator. Whatever is on the left gets assigned the value on the right! Python also comes with some handy syntax when using the assignment operator. 
                    
    x = 5   # x gets assigned the value 5
    y = x   # y gets assigned the value of x, which is 5

    x += 5  # x gets assigned the value of x + 5 (x = x + 5)

    print(x)
    print(y)

    # Output
    10
    5
  • Comparison operators are used to compare two variables or values. Comparison statements return booleans! 
                    
    print(5 == 5)   # Single equal signs are for assigning values, while double equal signs are for comparison!
    print(10 > 3)   # Greater than
    print(10 < 3)   # Less than
    print(10 >= 10) # Greater than or equal to
    print(10 <= 20) # Less than or equal to
    
    # Output
    True
    True
    False
    True
    False
  • Logical operators combine conditional statements. Their use cases are identical to their english counterparts! 
                    
    print(2 < 5 and 5 < 2)  # Returns True if both conditions are true
    print(2 < 5 or 5 < 2)   # Returns True if one condition is true
    print(5 < 2)            # Returns True if the condition is false

    # Output
    False
    True
    True
  • Just like with data types, Python has a bunch of other operators. Some, we'll encounter later on in the course. Others, well, you're going to find and learn them yourself!
