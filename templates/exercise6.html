Overview

  • You and I have registered and logged into websites before. Heck, you're using your own account in Syntax right now!
  • Lots of software with account features disallow duplicate usernames, along with other limitations.
  • In this exercise, you will implement a program that takes a username as input and looks at its validity.

Instructions

  • In Python, create a program that checks if a username is valid.
  • The program will take two inputs: username (string) and users (list).
  • The following rules should be checked in order:
  • Usernames are case sensitive. If a username is non-unique, print 'Invalid: Username already exists'
  • A username cannot have spaces. If a username has spaces, print 'Invalid: Username cannot have spaces'
  • A username must be a minimum of one (1) character and a maximum of fifteen (15) characters long. If a username does not fit this criteria, print 'Invalid: Username must be 1-15 characters long'
  • If a username is valid, print 'Valid'

Test Cases

  • You must print the validity of a username to the terminal line.
  • Your printed output must follow the given syntax.
  • You may not assume that a username is not empty.
