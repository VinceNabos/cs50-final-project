Introduction to Programming and Python

What is programming?

  • Most of us have taken computers for granted. But behind these seemingly magical boxes are complex processes built from ones and zeros.
  • One such process is programming. Essentially, programming is the process of taking some input and producing some output.
  • Computer programming is the process of writing instructions that tell a computer what to do with some input to produce some output.
  • Today, writing instructions, or so-called programs, uses programming languages so humans can interact with computers and ultimately solve problems.
  • One such language is Python.

What is Python?

  • Python is a popular programming language, known for its beginner-friendly syntax and readability.
  • Syntax is simply a set of rules that dictate how a programming language is written. Think of it like the grammar and punctuation rules of a language in the real world.
  • Python syntax is similar to the English language. Its syntax also allows developers to write programs with fewer lines of code than other languages.

Learning Python with Syntax

  • Syntax is a site aimed to teach the fundamentals of programming with Python.
  • It assumes that you have zero prior experience with computer science or programming.
  • Take your time to read and understand the contents of each lesson. Then, take your newfound knowledge out for a spin on the corresponding exercises, or even outside of Syntax!
  • With that out of the way, good luck on your programming journey!
