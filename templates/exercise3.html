Overview

  • FizzBuzz is a classic programming exercise aimed to teach operators, conditionals, and loops!
  • When learning any language, you can try solving FizzBuzz again to test if you got the fundamentals figured out!
  • In this exercise, you will write a FizzBuzz program in Python.

Instructions

  • The FizzBuzz function only has one argument: n.
  • Your task is to print a specific output for each number from 1-100.
  • If the number is divisible by 3 and 5, print 'FizzBuzz'.
  • If the number is divisible by 3, print 'Fizz'.
  • If the number is divisible by 5, print 'Buzx'.
  • Otherwise, print the number.

Test Cases

  • n will always be an integer valued 100.
  • It is fine to not use the variable n for your solution. However, your code must be inside the fizzbuzz() function
  • If your printed output is a string, it must follow the correct capitalization.
